Fitness Tracker App
A simple and responsive Fitness Tracker App built using HTML, CSS, and JavaScript. This application allows users to manually track their daily fitness activities, including workout type, workout duration, steps walked, and calories burned. All data is stored locally using the browser's Local Storage, ensuring records remain available even after refreshing the page.
📌 Features
Add daily fitness records
Track exercise type, workout duration, steps, and calories burned
View all fitness records in a table
Delete unwanted records
Daily progress bars for steps and calories
Weekly summary showing total steps, workout time, and calories burned
Local Storage support (no database required)
Responsive and user-friendly interface

🚀 How to Run the Project
Download or clone this repository.
Open the project folder in Visual Studio Code.
Install the Live Server extension (optional).
Open index.html in your browser or launch it using Live Server.
Start adding your fitness records.
